# 客家金柚特色农产品AI智荐系统 — 使用说明书

## 一、项目简介

《客家金柚特色农产品AI智荐系统》是一个基于大模型（DeepSeek）与规则式融合推荐算法的梅州客家金柚数字化展示与智能选品平台。系统聚焦**AI智荐推荐、智能问答、内容生成**三大核心功能，不涉及任何商业交易与支付。

| 信息 | 内容 |
|------|------|
| 项目类型 | 大学生创新训练计划项目 |
| 所属院校 | 嘉应学院 |
| 开发周期 | 2026.03 — 2027.03 |
| 技术栈 | Vue 3 + 微信小程序 + Spring Boot + Python Flask + MySQL + Redis + DeepSeek API |

## 三、快速启动（5 步）

### 准备工作

确保计算机上已安装 **JDK 17+、Python 3.9+、Node.js 18+、Maven 3.6+**。MySQL 和 Redis 在生产模式必需，**演示模式（零依赖）**只需 JDK + Python + Node.js。

---

### 第 1 步：配置环境变量 + 初始化数据库

**配置环境变量**：复制 `.env.example` 为 `.env`，填入实际值：

```bash
cd Desktop/v1.0
cp .env.example .env
```

需要配置的关键变量：`DEEPSEEK_API_KEY`（必需）、`DB_PASSWORD`（MySQL 密码）、以及可选的 `REDIS_PASSWORD`、`WECHAT_APP_ID` 等。

**初始化数据库**（可选——跳过则使用第 3 步的演示模式）：

```bash
mysql -u root -p < init_db.sql
```

执行成功后，`golden_pomelo_ai` 数据库中将包含 5 张表：
- `sys_user` — 用户表
- `golden_pomelo_knowledge` — 金柚知识库（含初始数据）
- `pomelo_prompt_library` — Prompt 模板库（含 3 条示例模板）
- `algo_rule_params` — 算法参数表（含 10 条默认参数）
- `llm_invoke_log` — 大模型调用日志

---

### 第 2 步：启动 Python AI 算法层（端口 5000）

```bash
cd Desktop/v1.0

# 1. 安装 Python 依赖
pip install -r ai_layer/requirements.txt

# 2. 设置 DeepSeek API Key（替换为你的真实 Key）
#    Windows CMD:
set DEEPSEEK_API_KEY=sk-你的Key

#    Windows PowerShell:
$env:DEEPSEEK_API_KEY="sk-你的Key"

#    Linux / macOS:
export DEEPSEEK_API_KEY="sk-你的Key"

# 3. 启动 Flask 服务
python run.py
```

验证：浏览器打开 http://127.0.0.1:5000/api/health

看到 `{"code":200,"data":{"status":"running"}}` 即表示成功。

---

### 第 3 步：启动 Spring Boot 后端（端口 8080）

**方式 A：演示模式（推荐，零外部依赖）**

无需 MySQL 或 Redis——使用 H2 内存数据库：

```bash
cd Desktop/v1.0/backend

# Windows CMD:
mvn spring-boot:run -Dspring-boot.run.profiles=demo

# Linux / macOS:
./mvnw spring-boot:run -Dspring-boot.run.profiles=demo
```

**方式 B：完整模式（需要 MySQL + Redis）**

```bash
cd Desktop/v1.0/backend
mvn spring-boot:run
```

启动成功后，控制台会打印 Spring Boot logo 以及 `AI层连接成功` 的日志。

> **注意**：如果 MySQL 密码不是空，请在 `src/main/resources/application.properties` 中修改：
> ```properties
> spring.datasource.password=你的数据库密码
> ```

---

### 第 4 步：启动 Vue 3 Web 前端（端口 3000，推荐）

```bash
cd Desktop/v1.0/web

# 1. 安装依赖（仅首次）
npm install

# 2. 启动开发服务器
npm run dev
```

浏览器打开 **http://localhost:3000** 即可使用系统。

Web 前端自动将 API 请求代理到 `http://127.0.0.1:8080`，无需额外配置。

---

### 第 5 步：启动微信小程序（可选）

1. 打开**微信开发者工具**
2. 选择「导入项目」→ 目录选择 `Desktop/123/miniapp`
3. 填写 AppID（可使用测试号）
4. 在菜单「详情 → 本地设置」中勾选 **「不校验合法域名」**

## 四、Web 前端使用指南

启动后在浏览器打开 `http://localhost:3000`，看到以下页面：

### 4.1 首页

页面顶部为导航栏，包含**首页 / AI智荐 / 文案生成 / 我的 / 登录**五个入口。

首页包含三个区域：

| 区域 | 功能 |
|------|------|
| **Hero 入口** | 点击「开始 AI 智荐」进入聊天页；点击「生成文案」进入文案页 |
| **热门提问** | 点击预设问题快速跳转到 AI 智荐并自动发送 |
| **金柚精选** | 展示知识库中的金柚卡片，点击可查看详情 |

### 4.2 AI 智荐（核心页面）

页面顶部有两个模式切换按钮：

- **🛒 选购推荐**：输入选购需求（如"200 元中秋送礼客家亲友"），AI 返回个性化推荐 + 金柚卡片
- **💬 知识问答**：输入知识问题（如"金柚怎么保存"），AI 返回专业回答

操作步骤：
1. 选择模式（选购推荐 / 知识问答）
2. 在底部输入框中输入需求或问题
3. 按 Enter 或点击「发送」按钮
4. 等待 AI 响应（约 3-5 秒）
5. 如果是选购推荐，会展示金柚卡片（含名称、产地、评分、推荐理由）
6. 点击金柚卡片可进入详情页

### 4.3 文案生成

1. 选择场景：「📱 朋友圈推文」或「🛒 电商详情页」
2. 填写金柚品名（如"梅县松口沙田柚"）
3. 可选：填写自定义要求（如"突出中秋送礼"）
4. 点击「生成文案」
5. 查看生成结果，点击「复制文案」复制到剪贴板

### 4.4 金柚详情页

点击任意金柚卡片即跳转到详情页，展示完整信息：
- 产地、品类、规格、价格、重量、季节
- 口感风味、种植工艺
- 客家文化关联
- 食用搭配、保鲜储藏
- 基础评分

点击底部「向 AI 了解更多」可跳转到聊天页自动提问。

### 4.5 个人中心

- 登录后显示用户信息和**近期提问记录**
- 点击历史提问可跳转到 AI 智荐重新提问
- 点击「管理后台」进入管理页面

### 4.6 管理后台

管理员可在此管理三个模块：

| Tab | 功能 |
|------|------|
| 📚 知识库 | 查看/新增/编辑/删除金柚知识条目 |
| 💬 Prompt 库 | 查看/新增/编辑/删除 Prompt 模板 |
| 📊 算法参数 | 在线修改三维度权重 + 规则/LLM 融合比例 |

## 六、管理员操作指南

### 6.1 管理员权限

- 默认 `userId=1` 的用户拥有管理员权限
- 管理员可操作知识库/Prompt 库的增删改
- 管理员可在线修改算法参数

### 6.2 添加金柚知识条目

1. 进入管理后台（页面右上角「我的」→「管理后台」）
2. 选择「📚 知识库」Tab
3. 点击「+ 新增」按钮
4. 填写表单：
   - **金柚品名**（必填）：如"梅县松口沙田柚"
   - **品类**：沙田柚 / 蜜柚 / 文旦柚
   - **产地**：如"梅州市梅县区松口镇"
   - **价格区间**：如"88-128元/箱"
   - **口感描述**：如"清甜化渣、蜜香浓郁"
5. 点击「保存」

### 6.3 编辑 Prompt 模板

1. 进入管理后台 →「💬 Prompt 库」Tab
2. 点击目标模板的「编辑」
3. 修改模板内容（支持 `{{user_input}}` 占位符）
4. 保存

### 6.4 调整推荐算法权重

1. 进入管理后台 →「📊 算法参数」Tab
2. 修改各权重值：
   - **三维度权重**（需求匹配度 + 场景适配度 + 客家特色）：之和应为 1.0
   - **融合权重**（规则分 + LLM 分）：之和应为 1.0
3. 修改立即生效，下次推荐请求将使用新权重

## 九、常见问题

### Q1：启动 Flask 时报 "No module named 'xxx'"

```bash
pip install -r ai_layer/requirements.txt
```

### Q2：Spring Boot 启动报数据库连接失败

**方案 A**：检查 MySQL 是否已启动，确认 `application.properties` 中的数据库用户名和密码是否正确。

**方案 B**：使用演示模式（无需 MySQL）：
```bash
mvn spring-boot:run -Dspring-boot.run.profiles=demo
```

### Q3：AI 智荐返回 "抱歉" 或无响应

检查 DeepSeek API Key 是否正确设置：
```bash
echo $DEEPSEEK_API_KEY   # Linux/macOS
echo %DEEPSEEK_API_KEY%  # Windows CMD
```

### Q4：Web 前端请求报 404

确认 Spring Boot (8080) 和 Flask (5000) 都已启动。Web 前端会自动代理 `/api` 到 8080。

### Q5：微信小程序无法请求后端

在微信开发者工具中勾选「详情 → 本地设置 → 不校验合法域名」。

### Q6：知识库/Prompt 管理操作失败（403）

需要使用管理员账号登录。默认 `userId=1` 为管理员。

### Q7：如何修改 DeepSeek API Key

修改环境变量 `DEEPSEEK_API_KEY` 后**重新启动** Flask 服务即可生效。

### Q8：如何不安装 MySQL/Redis 直接跑起来？

使用演示模式——只需 JDK 17+ + Python 3.9+ + Node.js 18+：

```bash
# 终端 1：启动 AI 层
set DEEPSEEK_API_KEY=sk-你的key
python run.py

# 终端 2：启动后端（演示模式）
cd backend
mvn spring-boot:run -Dspring-boot.run.profiles=demo

# 终端 3：启动前端
cd web
npm install
npm run dev
```

浏览器打开 http://localhost:3000 即可体验全部功能（知识库预置 7 条示例金柚数据）。

### Q9：如何运行测试？

```bash
# 安装测试依赖
pip install pytest

# 运行全部 46 个单元测试
python -m pytest ai_layer/tests/ -v
```
